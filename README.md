# 📚 Estudos em Java

Repositório dedicado aos meus estudos em Java durante a graduação em Ciências da Computação.
Aqui concentro exercícios, práticas e exemplos desenvolvidos ao longo das aulas, com foco na construção de uma base sólida em lógica de programação e desenvolvimento orientado a objetos.

---

## 🎯 Objetivo

Este repositório tem como objetivo documentar minha evolução na linguagem Java, consolidando conceitos fundamentais e boas práticas de desenvolvimento.
Além disso, serve como apoio para revisões futuras e como portfólio inicial na área de tecnologia.

---

## 🛠️ Tecnologias utilizadas

- Java
- IDE (NetBeans / IntelliJ)

---

## 📂 Estrutura do repositório

Os códigos estão organizados por temas e conteúdos abordados em aula:

- Arrays e matrizes
- Desvio condicional
- Estruturas de repetição

---

## 🚀 Exemplos de exercícios

- Validação de dados
- Ordenação (ex: Bubble Sort)
- Operações com matrizes
- Sistemas simples via console

---

## 📈 Evolução

Este repositório será atualizado continuamente conforme avanço nos estudos, incorporando novos conceitos e desafios.
